Explore bikeshare data

Data from 3 cities on bikeshare rentals over the first 6 months of 2017.
Data provided by Motivate: https://www.motivateco.com/

The data is presented in the form of an interactive web app using the Python framework Flask.

The app can be executed in the terminal and served locally using Flask's developer mode as follows:

1) In the terminal, type: python bikeshare.py
2) Navigate to http://127.0.0.1:5000 in your browser
